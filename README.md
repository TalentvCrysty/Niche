Pull Request Bot

Setup

1) Create a .env file with:

GITHUB_TOKEN=
GITHUB_OWNER=
GITHUB_REPO=
DEFAULT_BRANCH=main
WORKING_BRANCH_PREFIX=bot/
SCHEDULE_SECONDS=1800

2) Install and run:

npm install
npm run dev

Production

npm run build
npm start
